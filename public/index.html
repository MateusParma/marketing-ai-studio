Instagram & Facebook

Via Meta Graph API. Publique posts com imagem diretamente.

Não conectado

Como conectar (3 passos):

  1. Acesse o Explorador da Graph API
  2. No topo, selecione seu app (itime marketing)
  3. Clique em "Gerar token de acesso" — marque as permissões: pages_manage_posts, pages_read_engagement, instagram_basic, instagram_content_publish
  4. Copie o token gerado e cole abaixo
  5. O Page ID da sua Página do Facebook (vá na sua Página → Sobre → role até ver o ID numérico, ou use a Graph API: /me/accounts)

Wix Blog

Publique posts diretamente no blog do seu site Wix.

Não conectado

Para conectar:

  1. Acesse manage.wix.com
  2. Vá em Configurações → Chaves de API
  3. Crie uma chave com permissão "Blog" (leitura e escrita)
  4. Copie a API Key e o Site ID
✅ Wix conectado
